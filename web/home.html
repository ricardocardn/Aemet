GC Meteorological API

V1.0

You can use this API to obtain information about temperatures in Gran Canaria among days. All data has been obtained from Aemet. In the box bellow, you can see the paths you need to use according to reach your purpose. Any help you may need is also included after this paragraph.

  • To see all max temperatures among days -> /v1/html/places/with-max-temperature?from={:date}
  • To see all min temperatures among days -> /v1/html/places/with-min-temperature?from={:date}
  • To to check today's max temperature among days -> /v1/html/places/with-max-temperature
  • To to check today's min temperature among days -> /v1/html/places/with-min-temperature
  • To to check max temperatures between two days -> /v1/html/places/with-max-temperature?from={:date1}&to={:date2}
  • To to check min temperatures between two days -> /v1/html/places/with-min-temperature?from={:date1}&to={:date2}
  • br
  • (*) date param must have the next format: dd-MM-yyyy

This API uses a datamart created from Aemet data and is updated every hour. Thus, max. or min. temperature could change.
